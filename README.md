# LeetStats Printer

A simple web application that fetches and displays LeetCode statistics for a user in a clean and readable format.

This project allows users to enter their LeetCode username and view their problem-solving statistics such as solved problems and other relevant data in a visually structured format.

---

## 🚀 Features

- Fetch LeetCode user statistics
- Display stats in a clean UI
- Lightweight frontend project
- Simple HTML, CSS, and JavaScript implementation
- Easy to run locally

---

## 🛠️ Tech Stack

- **HTML** – Structure of the application  
- **CSS** – Styling and layout  
- **JavaScript** – Fetching and displaying LeetCode data
